# lxp-scan Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A Rust CLI (`lxp-scan`) that answers cross-repo impact ("who uses symbol X, with which props") and version-drift questions over the FE repos in `~/Leapxpert/FE`.

**Architecture:** Library crate + thin binary. Per repo: discover (`package.json`), walk source files (`ignore`), resolve import specifiers against `tsconfig.json` aliases, analyze files with a substring prefilter followed by an `oxc_parser` AST pass, all parallelized with `rayon`. Rendering (table/JSON) is isolated in `report.rs`.

**Tech Stack:** Rust, clap (derive), oxc_parser/oxc_ast/oxc_allocator/oxc_span, ignore, rayon, serde/serde_json, jsonc-parser, comfy-table, anyhow.

**Spec:** `docs/superpowers/specs/2026-07-13-lxp-scan-design.md`

**API-drift caveat:** The oxc crates evolve quickly. Code below targets the API shape current as of writing; if a pinned version's names differ (e.g. visitor method names, `JSXElementName` variants), adapt per that version's docs.rs rather than fighting the plan. Everything else (clap, serde, ignore, rayon) is stable.

```rust
pub fn discover_repos(root: &Path, warnings: &mut Vec<String>) -> Result<Vec<Repo>> {
    let entries = std::fs::read_dir(root)
        .map_err(|e| anyhow::anyhow!("cannot read root {}: {e}", root.display()))?;
    let mut repos = Vec::new();
    for entry in entries.flatten() {
        let dir = entry.path();
        if !dir.is_dir() {
            continue;
        }
        let pkg = dir.join("package.json");
        if !pkg.is_file() {
            continue;
        }
        let name = entry.file_name().to_string_lossy().into_owned();
        let text = match std::fs::read_to_string(&pkg) {
            Ok(t) => t,
            Err(e) => {
                warnings.push(format!("{name}: cannot read package.json: {e}"));
                continue;
            }
        };
        let value: serde_json::Value = match serde_json::from_str(&text) {
            Ok(v) => v,
            Err(e) => {
                warnings.push(format!("{name}: malformed package.json: {e}"));
                continue;
            }
        };
        let mut deps = BTreeMap::new();
        for key in ["dependencies", "devDependencies"] {
            if let Some(map) = value.get(key).and_then(|v| v.as_object()) {
                for (k, v) in map {
                    if let Some(ver) = v.as_str() {
                        deps.insert(k.clone(), ver.to_string());
                    }
                }
            }
        }
        repos.push(Repo { name, root: dir, deps });
    }
    repos.sort_by(|a, b| a.name.cmp(&b.name));
    Ok(repos)
}
```

```rust
pub fn compute_drift(repos: &[Repo]) -> Vec<DriftRow> {
    let mut by_package: BTreeMap<String, BTreeMap<String, String>> = BTreeMap::new();
    for repo in repos {
        for (pkg, ver) in &repo.deps {
            if TRACKED_PREFIXES.iter().any(|p| pkg.starts_with(p)) {
                by_package
                    .entry(pkg.clone())
                    .or_default()
                    .insert(repo.name.clone(), ver.clone());
            }
        }
    }
    by_package
        .into_iter()
        .map(|(package, versions)| {
            let parsed: Vec<_> = versions.values().filter_map(|v| major_minor(v)).collect();
            let level = if parsed.windows(2).any(|w| w[0].0 != w[1].0) {
                DriftLevel::Major
            } else if parsed.windows(2).any(|w| w[0].1 != w[1].1) {
                DriftLevel::Minor
            } else {
                DriftLevel::Same
            };
            DriftRow { package, versions, level }
        })
        .collect()
}
```

```rust
pub fn drift_table(rows: &[DriftRow], repo_names: &[String]) -> String {
    let mut table = Table::new();
    let mut header = vec!["package".to_string()];
    header.extend(repo_names.iter().cloned());
    header.push("drift".to_string());
    table.set_header(header);
    for row in rows {
        let mut cells = vec![row.package.clone()];
        for repo in repo_names {
            cells.push(row.versions.get(repo).cloned().unwrap_or_else(|| "-".to_string()));
        }
        cells.push(format!("{:?}", row.level));
        table.add_row(cells);
    }
    table.to_string()
}
```

```rust
pub fn source_files(repo_root: &Path) -> Vec<PathBuf> {
    let mut out = Vec::new();
    let walker = ignore::WalkBuilder::new(repo_root)
        .hidden(true)
        .git_ignore(true)
        .filter_entry(|entry| {
            let name = entry.file_name().to_string_lossy();
            !SKIP_DIRS.contains(&name.as_ref())
        })
        .build();
    for entry in walker.flatten() {
        let path = entry.path();
        if !path.is_file() {
            continue;
        }
        let Some(ext) = path.extension().and_then(|e| e.to_str()) else {
            continue;
        };
        if !EXTENSIONS.contains(&ext) {
            continue;
        }
        if path.to_string_lossy().ends_with(".d.ts") {
            continue;
        }
        out.push(path.to_path_buf());
    }
    out.sort();
    out
}
```

```rust
impl RepoResolver {
    pub fn from_repo(repo_root: &Path, warnings: &mut Vec<String>) -> Self {
        let mut resolver = RepoResolver {
            repo_root: repo_root.to_path_buf(),
            base_url: None,
            paths: Vec::new(),
        };
        let tsconfig = repo_root.join("tsconfig.json");
        let text = match std::fs::read_to_string(&tsconfig) {
            Ok(t) => t,
            Err(_) => {
                warnings.push(format!(
                    "{}: no tsconfig.json, alias resolution disabled",
                    repo_root.display()
                ));
                return resolver;
            }
        };
        // JSONC: tsconfig may contain comments and trailing commas.
        let value = match jsonc_parser::parse_to_serde_value(&text, &Default::default()) {
            Ok(Some(v)) => v,
            _ => {
                warnings.push(format!(
                    "{}: unreadable tsconfig.json, alias resolution disabled",
                    tsconfig.display()
                ));
                return resolver;
            }
        };
        let opts = value.get("compilerOptions").cloned().unwrap_or_default();
        let base = opts.get("baseUrl").and_then(|v| v.as_str()).map(|b| repo_root.join(b));
        if let Some(paths) = opts.get("paths").and_then(|v| v.as_object()) {
            let alias_base = base.clone().unwrap_or_else(|| repo_root.to_path_buf());
            for (pattern, targets) in paths {
                let pattern = pattern.trim_end_matches('*').to_string();
                let targets = targets
                    .as_array()
                    .map(|arr| {
                        arr.iter()
                            .filter_map(|t| t.as_str())
                            .map(|t| alias_base.join(t.trim_end_matches('*')))
                            .collect()
                    })
                    .unwrap_or_default();
                resolver.paths.push((pattern, targets));
            }
        }
        resolver.base_url = base;
        resolver
    }

    pub fn resolve(&self, importer: &Path, specifier: &str) -> ResolvedImport {
        if specifier.starts_with('.') {
            let joined = importer.parent().unwrap_or(Path::new("/")).join(specifier);
            let normalized = normalize(&joined);
            return ResolvedImport::File(resolve_extension(&normalized).unwrap_or(normalized));
        }
        for (pattern, targets) in &self.paths {
            if let Some(rest) = specifier.strip_prefix(pattern.as_str()) {
                for target in targets {
                    let candidate = normalize(&target.join(rest));
                    if let Some(found) = resolve_extension(&candidate) {
                        return ResolvedImport::File(found);
                    }
                }
            }
        }
        if let Some(base) = &self.base_url {
            let candidate = normalize(&base.join(specifier));
            if let Some(found) = resolve_extension(&candidate) {
                return ResolvedImport::File(found);
            }
        }
        ResolvedImport::Package(specifier.to_string())
    }

    pub fn display(&self, resolved: &ResolvedImport) -> String {
        match resolved {
            ResolvedImport::Package(s) => s.clone(),
            ResolvedImport::File(p) => p
                .strip_prefix(&self.repo_root)
                .unwrap_or(p)
                .to_string_lossy()
                .into_owned(),
        }
    }
}

fn normalize(path: &Path) -> PathBuf {
    let mut out = PathBuf::new();
    for comp in path.components() {
        match comp {
            std::path::Component::ParentDir => {
                out.pop();
            }
            std::path::Component::CurDir => {}
            other => out.push(other),
        }
    }
    out
}

fn resolve_extension(base: &Path) -> Option<PathBuf> {
    const EXTS: [&str; 4] = ["ts", "tsx", "js", "jsx"];
    if base.is_file() {
        return Some(base.to_path_buf());
    }
    for ext in EXTS {
        let candidate = base.with_extension(ext);
        if candidate.is_file() {
            return Some(candidate);
        }
    }
    for ext in EXTS {
        let candidate = base.join(format!("index.{ext}"));
        if candidate.is_file() {
            return Some(candidate);
        }
    }
    None
}
```

```rust
use oxc_allocator::Allocator;
use oxc_ast::ast::*;
use oxc_ast::Visit;
use oxc_parser::Parser;
use oxc_span::SourceType;

struct UsageCollector<'s> {
    local: &'s str,
    refs: usize,
    jsx_uses: usize,
    jsx_props: BTreeSet<String>,
}

impl<'a, 's> Visit<'a> for UsageCollector<'s> {
    fn visit_identifier_reference(&mut self, ident: &IdentifierReference<'a>) {
        if ident.name == self.local {
            self.refs += 1;
        }
    }

    fn visit_jsx_opening_element(&mut self, el: &JSXOpeningElement<'a>) {
        if let JSXElementName::Identifier(id) = &el.name {
            if id.name == self.local {
                self.jsx_uses += 1;
                for attr in &el.attributes {
                    if let JSXAttributeItem::Attribute(a) = attr {
                        if let JSXAttributeName::Identifier(n) = &a.name {
                            self.jsx_props.insert(n.name.to_string());
                        }
                    }
                }
            }
        }
        // keep walking children (attribute expressions may contain JSX)
        oxc_ast::visit::walk::walk_jsx_opening_element(self, el);
    }
}

fn line_of(text: &str, byte_offset: usize) -> usize {
    text[..byte_offset.min(text.len())].bytes().filter(|b| *b == b'\n').count() + 1
}

pub fn analyze_file(path: &Path, source_text: &str, symbol: &str) -> anyhow::Result<Vec<FileFindings>> {
    let allocator = Allocator::default();
    let source_type = SourceType::from_path(path)
        .map_err(|e| anyhow::anyhow!("{}: unsupported source type: {e:?}", path.display()))?;
    let ret = Parser::new(&allocator, source_text, source_type).parse();
    if ret.panicked || !ret.errors.is_empty() && ret.program.body.is_empty() {
        anyhow::bail!("{}: parse failed", path.display());
    }

    // Pass 1: find matching imports -> (local_name, source, line)
    let mut matches: Vec<(String, String, usize)> = Vec::new();
    for stmt in &ret.program.body {
        let Statement::ImportDeclaration(decl) = stmt else { continue };
        let Some(specifiers) = &decl.specifiers else { continue };
        for spec in specifiers {
            let hit = match spec {
                ImportDeclarationSpecifier::ImportSpecifier(named) => {
                    named.imported.name() == symbol
                }
                ImportDeclarationSpecifier::ImportDefaultSpecifier(default) => {
                    default.local.name == symbol
                }
                ImportDeclarationSpecifier::ImportNamespaceSpecifier(_) => false, // v1 limitation
            };
            if hit {
                let local = match spec {
                    ImportDeclarationSpecifier::ImportSpecifier(named) => named.local.name.to_string(),
                    ImportDeclarationSpecifier::ImportDefaultSpecifier(default) => default.local.name.to_string(),
                    _ => unreachable!(),
                };
                matches.push((
                    local,
                    decl.source.value.to_string(),
                    line_of(source_text, decl.span.start as usize),
                ));
            }
        }
    }
    if matches.is_empty() {
        return Ok(Vec::new());
    }

    // Pass 2: per matching local binding, collect usage
    let mut out = Vec::new();
    for (local, source, line) in matches {
        let mut collector = UsageCollector {
            local: &local,
            refs: 0,
            jsx_uses: 0,
            jsx_props: BTreeSet::new(),
        };
        collector.visit_program(&ret.program);
        // visit_identifier_reference does not fire for the import declaration
        // itself (binding identifiers are not references), so no correction needed.
        out.push(FileFindings {
            line,
            source,
            refs: collector.refs,
            jsx_uses: collector.jsx_uses,
            jsx_props: collector.jsx_props,
        });
    }
    Ok(out)
}
```

```rust
pub fn run_impact(
    root: &Path,
    symbol: &str,
    from: Option<&str>,
    warnings: &mut Vec<String>,
) -> Result<Vec<ImpactHit>> {
    let repos = discover_repos(root, warnings)?;
    let mut hits = Vec::new();
    for repo in &repos {
        let resolver = RepoResolver::from_repo(&repo.root, warnings);
        let files = source_files(&repo.root);
        let (repo_hits, repo_warnings): (Vec<_>, Vec<_>) = files
            .par_iter()
            .map(|file| -> (Vec<ImpactHit>, Option<String>) {
                let Ok(text) = std::fs::read_to_string(file) else {
                    return (Vec::new(), Some(format!("{}: unreadable", file.display())));
                };
                // cheap prefilter: skip the AST entirely when the symbol string is absent
                if !text.contains(symbol) {
                    return (Vec::new(), None);
                }
                match analyzer::analyze_file(file, &text, symbol) {
                    Ok(findings) => {
                        let file_rel = file
                            .strip_prefix(&repo.root)
                            .unwrap_or(file)
                            .to_string_lossy()
                            .into_owned();
                        let hits = findings
                            .into_iter()
                            .filter_map(|f| {
                                let resolved = resolver.resolve(file, &f.source);
                                let display = resolver.display(&resolved);
                                if let Some(filter) = from {
                                    if !display.contains(filter) {
                                        return None;
                                    }
                                }
                                Some(ImpactHit {
                                    repo: repo.name.clone(),
                                    file: file_rel.clone(),
                                    line: f.line,
                                    source: display,
                                    refs: f.refs,
                                    jsx_uses: f.jsx_uses,
                                    jsx_props: f.jsx_props,
                                })
                            })
                            .collect();
                        (hits, None)
                    }
                    Err(e) => (Vec::new(), Some(format!("{e:#}"))),
                }
            })
            .unzip();
        hits.extend(repo_hits.into_iter().flatten());
        warnings.extend(repo_warnings.into_iter().flatten());
    }
    hits.sort_by(|a, b| (&a.repo, &a.file, a.line).cmp(&(&b.repo, &b.file, b.line)));
    Ok(hits)
}
```
